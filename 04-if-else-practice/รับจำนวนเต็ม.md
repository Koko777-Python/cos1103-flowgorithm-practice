# รับจำนวนเต็ม

[← กลับหน้าหลัก](../README.md) · [ดาวน์โหลดไฟล์ Flowgorithm](<./รับจำนวนเต็ม.fprg>)

## โจทย์

รับจำนวนเต็ม 1 ค่า แล้วตรวจสอบว่าเป็นจำนวนบวก หรือเป็นศูนย์/จำนวนลบ

**แนวคิดที่ฝึก:** การเลือกทำงาน 2 ทางด้วย `If...Else`

## ผังงานจาก Flowgorithm

![ผังงาน รับจำนวนเต็ม](<./images/รับจำนวนเต็ม.png>)

> ภาพหน้าจอนี้จับคู่กับไฟล์ `รับจำนวนเต็ม.fprg` ตามชื่อเดียวกัน

## Pseudocode

```text
เริ่มต้น
    ประกาศ Integer number
    แสดงผล "Enter number"
    รับค่า number

    ถ้า number > 0 แล้ว
        แสดงผล "Positive"
    มิฉะนั้น
        แสดงผล "Zero or Negative"
    จบเงื่อนไข
จบการทำงาน
```

## ตัวอย่างทดสอบ

| ค่า `number` | ผลลัพธ์ |
|---:|---|
| `8` | `Positive` |
| `0` | `Zero or Negative` |
| `-3` | `Zero or Negative` |
